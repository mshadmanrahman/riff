# Riff - LinkedIn Engagement Assistant

**Read the room before you comment.**

Riff extracts LinkedIn posts and comments into clean, structured markdown. One click. Copy to clipboard. Paste into any AI tool to draft thoughtful replies.

## How It Works

1. Navigate to any LinkedIn post or scroll your feed
2. Click the Riff icon
3. "Extract Post + Comments"
4. "Copy to Clipboard"
5. Paste into Claude, ChatGPT, or any AI tool
6. Get a draft that actually references what people said

## Two Modes

- **REPLY mode** (your posts): Extracts all comments so you can draft replies
- **COMMENT mode** (others' posts): Extracts the full post + comments so you can add unique value

## What Gets Extracted

- Post author and headline
- Full post text
- All visible comments with author names, headlines, and timestamps
- Nested replies
- Post type (text, image, video, article, poll)

## Privacy

Zero data collection. Everything runs locally in your browser. No analytics, no tracking, no servers. Content only exists in your clipboard after you copy it.

[Full privacy policy](PRIVACY.md)

## Install

**Chrome Web Store** (recommended): [Get Riff](https://chrome.google.com/webstore/detail/riff)

**Manual install:**
1. Download or clone this repo
2. Open `chrome://extensions` in Chrome/Chromium
3. Enable "Developer mode"
4. Click "Load unpacked" and select the repo folder

## Why "Read the Room"?

The best LinkedIn comments add unique value. But with 30+ comments on every post, most people either skip reading them or end up saying what someone already said.

Riff gives you the full conversation in one click so your comment can reference specific points, add new angles, and actually stand out.
